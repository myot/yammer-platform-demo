yammer-platform-demo

Javascript client for using Yammer Embed as well as Javascript SDK

Download .zip Download .tar.gz View on GitHub

Welcome to GitHub Pages.

Before you start using the Javascript SDK, you will need client application token from Yammer. Sign up at Yammer http://www.yammer.com/client_applications. Then update the value in server.conf file you would create in the following step. 


$ git clone git@github.com:myot/yammer-platform-demo.git
$ cd yammer-platform-demo
$ cp server-conf.template server-conf
$ npm install
$ node server.js
